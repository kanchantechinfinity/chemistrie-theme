# Chemistrie — Shopify Theme

Luxury pharmacist-skincare theme. Homepage replicates the reference design
(GSAP + ScrollTrigger + Lenis scroll animations); multi-page store with
Collection, Ritual, Pharmacists, Our Story, Journal, Contact, plus cart,
search, and customer-account templates.

## Structure
Standard Online Store 2.0 theme: `layout/`, `templates/` (+ `templates/customers/`),
`sections/`, `snippets/`, `assets/`, `config/`, `locales/`.

## Deploy
Connected to Shopify via GitHub integration — pushes to the default branch
sync to the store automatically.
